# Tela de Login e Cadastro — Frontend

**Demo ao vivo:**  
https://mairefm.github.io/frontend-login-cadastro/

---

## Sobre o projeto
Projeto em **HTML, CSS e JavaScript** desenvolvido como modelo para as minhas aulas de Frontend no **Colégio Dom Bosco**.

O objetivo é apresentar aos alunos:
- Estruturação de layouts com HTML  
- Estilização moderna com CSS  
- Efeitos de transição  
- Formulário de Login e Cadastro  
- Organização de pastas em projetos Frontend  

É um ótimo exemplo para quem está começando na programação.

---

## Tecnologias usadas
- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts**
- **GitHub Pages** (deploy)

---

## Como visualizar
Basta acessar o link da demo ao vivo:  
https://mairefm.github.io/frontend-login-cadastro/
